## 🌌 System Architecture
The **DEPI Training Portal** is not just a landing page; it's a high-performance **Headless CMS Interface** built to centralize the learning journey of the ALX4_SWD8_S1 cohort. By utilizing Google Sheets as a real-time Data Matrix, the portal eliminates manual maintenance while delivering a premium user experience.

### 🛠 Core Stack & Integration
> **Frontend Intelligence:** Developed using Vanilla ES6+ JavaScript to ensure zero-dependency speed and maximum performance.
> **Design System:** Utility-first Tailwind CSS implementation with a custom Neon-Glassmorphism aesthetic.
> **Data Synchronization:** Real-time JSON fetching from Google Sheets with intelligent `localStorage` state management.

---

## ✨ Mission-Critical Features

### 🔍 Dynamic Roadmap Explorer
An interactive, nested tree structure that renders sessions and tasks dynamically. It reflects changes in the Google Sheet instantly, providing the team with the most up-to-date schedule.

### 🧠 Intelligent Caching Engine
The system implements a sophisticated caching layer. It stores data locally for instant initial loads and performs background synchronization to ensure data integrity without blocking the UI.

### 📂 Centralized Resource Matrix
A unified dashboard for all DEPI assets:
- **Google Drive:** Direct access to all session materials.
- **SharePoint:** Automated links to technical recordings and task submission portals.
- **Mission Briefings:** Detailed task descriptions with integrated protocol checklists.
